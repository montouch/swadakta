Virtual work grading

Skills get checked differently.

Backend developer work is not graded like delivery work. Swadakta should score identity, portfolio evidence, communication, test task quality, milestone proof, and client review.

Identity + provenance

Real profile, verified ID, work history, honest photo.

Skill evidence

Portfolio, repo samples, references, certificates.

Test task

Small scoped milestone before larger release.

Acceptance proof

Files, demo, review notes, revision trail.

Safety line

Legal work only. ID verification before paid posting or receiving paid work. Proof before provider action. Swadakta does not hold client money or act as escrow.
